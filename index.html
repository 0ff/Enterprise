Enterprise: EFI Linux Loader

A Slightly Pompous Attempt to Boot Linux on Intel Macs

Enterprise (named after the Starship Enterprise from Star Trek), is an EFI program that is designed to assist in loading Linux distributions on USB sticks on Intel Macs, something that is continously regarded as being near to impossible due to EFI quirks.

The program is quite unique. It is written in C, and will be compiled to run on both 32-bit and 64-bit EFI firmware types. More information coming shortly!

It is designed to compliment Mac Linux USB Loader, though it can also be used separately (albeit with some setup work). The purpose of Enterprise is as the first stage in a two-stage booting process for Mac Linux USB Loader-created USB drives.
